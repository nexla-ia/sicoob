📋 INÍCIO RÁPIDO - 3 PASSOS

┌─────────────────────────────────────────────────────────────┐
│ PASSO 1: CRIAR TABELAS NO BANCO                             │
└─────────────────────────────────────────────────────────────┘

  psql -U postgres -d nexla_acesso -f backend/setup-database.sql

  OU abra o arquivo backend/setup-database.sql no pgAdmin/DBeaver


┌─────────────────────────────────────────────────────────────┐
│ PASSO 2: INICIAR BACKEND                                    │
└─────────────────────────────────────────────────────────────┘

  cd backend
  npm run start:dev

  Aguarde aparecer: 🚀 Backend rodando em http://localhost:3001


┌─────────────────────────────────────────────────────────────┐
│ PASSO 3: INICIAR FRONTEND (em outro terminal)              │
└─────────────────────────────────────────────────────────────┘

  npm run dev

  Acesse: http://localhost:5173
